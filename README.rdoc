== README

Artist-Paintings

Prerequisite:

* Ruby version: 2.3.8

* Rails version: 4.2.0

* Mysql database

Installation:

* fork this repository and bundle install all the gems.

* create a database named "mindfire"

* Migrate database by command 'rake db:migrate'

* run the local server by 'rails s'

* Open localhost:3000 on your browser.

* That's it for the installation:)
